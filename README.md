# StimCore — калькулятор глинокислотной обработки (ГКО)

Инженерное Streamlit-приложение по методике «Проектирование кислотной обработки»
(Приложение В). ГКО — последовательная обработка: сначала СКР (HCl), затем ГКР
(HCl + HF). Результаты задач сохраняются в `st.session_state` и подхватываются
следующими шагами.

## Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Деплой на Streamlit Community Cloud

1. Запушьте репозиторий на GitHub.
2. На https://share.streamlit.io → **Create app** → выберите репозиторий.
3. **Main file path:** `app.py`. Deploy.

`requirements.txt` и `runtime.txt` (Python 3.11) подхватываются автоматически.

## Навигация (Задачи ГКО)

1. Целесообразность ГКО
2. Расход и давление нагнетания
3. Продавочная и вытесняющая жидкость
4. Изменение пористости
5. Эффективность ГКО

## О задачах

Сквозная цепочка проектирования ГКО: отбор скважины-кандидата → безопасный
режим закачки → объёмы жидкостей → прогноз роста пористости → дополнительная
нефть и экономический эффект.

### 1. Целесообразность ГКО (В.1/В.2)

Прежде чем закачивать кислоту — доказать, что обработка даст эффект. Скважина
проверяется по одиннадцати критериям В.1–В.14: резерв продуктивности
ОП = Кф/Кпот и дебита ОД = Qф/Qож (есть ли что улучшать), приёмистость,
поглощающие пласты и гидропроводность (дойдёт ли кислота до загрязнённых
интервалов), глинистость и лабораторные kms/kmg (сработает ли химия),
энергетика пласта kэнр = pпл/pгст (очистится ли скважина после). Карбонатность
Cк решает тип обработки: ≥ 3 % — СКО→ГКО, < 3 % — только ГКО. Коэффициент
охвата kв.о даёт схему: весь разрез или поинтервально. Дополнительно —
скин-фактор S по Дюпюи: загрязнена ПЗП (S > 0) или нет (S < 0).
**Результат:** вердикт «пригодна/непригодна», тип обработки, схема охвата.

### 2. Расход и давление нагнетания (В.3)

С каким расходом и при каком давлении можно безопасно закачивать раствор,
чтобы не вызвать гидроразрыв и не повредить колонну. По кривой приёмистости
pу = f(t) пробного нагнетания определяется квазиустойчивое давление pк
(полка кривой). Проверки: qк ≥ qпр (В.10); pк ≤ pопр — иначе нельзя без
пакера (В.15); сравнение градиентов на забое grad pк = (pгст + pк)/(0,01·H)
против градиента гидроразрыва grad pгрп (В.16–В.20).
**Результат:** допустимые расход и давление + заключение об отсутствии ГРП.

### 3. Продавочная и вытесняющая жидкость (В.10)

Объёмы вспомогательных жидкостей, проталкивающих кислоту до пласта.
Вытесняющая: Vвтс = 1,2·Vз.рs по зоне растворения (В.56) или с запасом
Vвтс = 0,3·Vкс (В.57). Продавочная — геометрия НКТ и затруба:
Vпрд = 0,785·[dв²·Hн.о + (Dк² − dвн²)·(Hн.о − Hв.о)] (В.58).
**Результат:** Vвтс и Vпрд в м³ — сколько жидкости готовить на операцию.

### 4. Изменение пористости (В.15)

Прогноз роста пористости песчаного коллектора после ГКР, когда прямых
лабораторных замеров kmg нет — только данные о растворённых карбонатах и
глинах. Прирост в объёмных долях DVg = ρп·DGg/ρск (В.82), пористость после
ГКР mg = ms + DVg (В.83), коэффициенты kmg = mg/ms (В.84) и kmsg = kms·kmg
(В.85) — те самые, что в задаче 1 проверяются по критериям В.4–В.5.
**Результат:** ожидаемая пористость mg и коэффициенты kmg, kmsg.

### 5. Эффективность ГКО (В.16)

Итоговая проверка деньгами: сколько дополнительной нефти даст обработка и
окупится ли она. Приток через две зоны улучшенной проницаемости (СКР и ГКР)
даёт рост дебита Ag (В.86); новый дебит с учётом охвата Qg = Ag·εот·Qф +
(1−εот)·Qф (В.89); дополнительная нефть за период с поправкой на
обводнённость ΔQн (В.90); экономика Эн = (Цн − Сн)·ΔQн − Zко (В.91).
**Результат:** Ag, Qg, ΔQн и Эн по вариантам объёма кислоты — выбор
оптимального объёма обработки.
